📌 Description

This is a simple and responsive calculator web application built using HTML, CSS, and JavaScript.
It performs basic arithmetic operations with a clean and modern user interface.

This project is part of my learning journey in web development and helps me understand DOM manipulation, event handling, and UI design.

🚀 Features
Basic operations: ➕ Addition, ➖ Subtraction, ✖️ Multiplication, ➗ Division
Decimal number support
Clear button (C) to reset everything
Toggle sign (±) functionality
Responsive and modern UI design
Error handling (like division by zero)
🛠️ Technologies Used
HTML5 – Structure of the app
CSS3 – Styling and layout (Grid + Gradient UI)
JavaScript – Logic and functionality
